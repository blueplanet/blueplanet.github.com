OS 10.8 Gem Install Json Faild

Oct 4th, 2012

環境

  • Mac OS 10.8.2
  • ruby 1.9.2p320
  • gem 1.8.24
  • xcode 4.5 (4G182)

問題

gem install jsonを実行する場合、下記のエラーで落ちる

解決

  1. xcode でCommand Line Toolsをインストール
  2. sudo ln -sf /usr/bin/llvm-gcc-4.2 /usr/bin/gcc-4.2を実行
